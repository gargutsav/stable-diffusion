[Stable Diffusion](#stable-diffusion-v1) is a latent text-to-image diffusion
model.
Thanks to a generous compute donation from [Stability AI](https://stability.ai/) and support from [LAION](https://laion.ai/), we were able to train a Latent Diffusion Model on 512x512 images from a subset of the [LAION-5B](https://laion.ai/blog/laion-5b/) database. 
Similar to Google's [Imagen](https://arxiv.org/abs/2205.11487), 
this model uses a frozen CLIP ViT-L/14 text encoder to condition the model on text prompts.
With its 860M UNet and 123M text encoder, the model is relatively lightweight and runs on a GPU with at least 10GB VRAM.
See [this section](#stable-diffusion-v1) below and the [model card](https://huggingface.co/CompVis/stable-diffusion).

  
## Requirements
A suitable [conda](https://conda.io/) environment named `ldm` can be created
and activated with:

```
conda env create -f environment.yaml
conda activate ldm
```

You can also update an existing [latent diffusion](https://github.com/CompVis/latent-diffusion) environment by running

```
conda install pytorch torchvision -c pytorch
pip install transformers==4.19.2 diffusers invisible-watermark
pip install -e .
``` 

## Running inference
Place all prompts in a text file with one prompt per line. Then run the following command:

```
    python scripts/txt2img.py --n_samples <batch_size> --plms --ckpt <path to checkpoint> \
    --outdir <path to output folder> \
    --skip_grid \
    --H 512 \
    --W 512 \
    --from-file <path to prompts file>
```
Another parameter that can be experimented with is `scale` in `scripts/txt2img.py`, this controls the classifier-free guidance.

Once, the images are generated, we can evaluate metrics using [torch-fidelity](https://github.com/toshas/torch-fidelity). For this use 2-5k images for a good estimate. Two metrics can be calculated using the following commands:

#### Inception Score
```
fidelity --gpu 0 --isc --input1 <folder with generated images>
```

#### FID Score
```
fidelity --gpu 0 --fid --input1 <folder with generated images> --input2 <folder with real images>
```
